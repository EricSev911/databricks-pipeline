# CMS Medicare Inpatient + Outpatient Pipeline

Databricks medallion pipeline (Bronze / Silver / Gold) joining two public CMS Medicare datasets on provider CCN, with a Kaiser Permanente California focus and San Diego market drill-down.

**[View the notebook in .html format with full output](https://ericsev911.github.io/databricks-pipeline/cms_multi_dataset_pipeline.html)** - includes all tables, charts, and commentary

**[Browse the .py source code](https://github.com/EricSev911/databricks-pipeline/blob/main/cms_multi_dataset_pipeline.py)**

To run it: open the [source code](https://github.com/EricSev911/databricks-pipeline/blob/main/cms_multi_dataset_pipeline.py) and click the **Download raw file** button, import the file into a Databricks workspace, and hit Run All. The notebook creates its own schema and downloads the source data automatically.
